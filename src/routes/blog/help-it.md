# {title}

Template to create a blog with SvelteKit

### Get Started

To create a new project based on this template using degit:

```bash
npx degit el3um4s/memento-sveltekit-and-github-pages
```

Then install the dependencies with

```bash
npm install
```

### Command

- `npm run dev`
- `npm run build`
- `npm run deploy`
